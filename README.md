# password-based-circuit-breaker
Embedded password-authenticated circuit breaker system using Arduino and relay-based protection control.

An Arduino-based password protected circuit breaker system designed to control electrical circuit operation through secure authentication and embedded relay switching.

## Features
- Password-based authentication
- Embedded control using Arduino
- Relay-based circuit switching
- Unauthorized access prevention
- LCD/keypad integration
- Electrical protection control

## Components Used
- Arduino Uno/Nano
- 4x4 Keypad
- Relay Module
- LCD Display
- Buzzer
- Power Supply
- Connecting Wires

## Working Principle
The system uses an Arduino microcontroller to verify a user-entered password through a keypad interface. Upon successful authentication, the relay activates and allows circuit operation. Incorrect password attempts keep the circuit disconnected, improving safety and access control.

- Restricted electrical systems
- Industrial control panels
- Educational embedded systems projects
- Smart electrical protection systems
- Secure appliance control
